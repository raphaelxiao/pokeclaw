# TTS 文本预处理规则

参见 `core/tts_preprocess.py` 中的 `preprocess_for_tts()` 函数。

## 结构化内容
| 规则 | 转换 |
|------|------|
| MD 表格 | → `此处我整理了表格，可以在屏幕阅读。` |
| Bullet 列表 | → `第一，…。第二，…。第三，…。` |

## 符号替换
| 原始 | 替换 |
|------|------|
| `～` | `至` |
| `>` `＞` | `大于` |
| `<` `＜` | `小于` |
| `=` `＝` | `等于` |
| `≠` | `不等于` |
| `+` `＋` | `加` |
| `→` | `至` |
| `——` | `，` |

## 单位
| 原始 | 替换 |
|------|------|
| `nm` | `纳米` |
| `GWh` | `吉瓦时` |
| `Wh/kg` | `瓦时每千克` |
| `数字+L` | `数字+升` |

## 数字处理
| 规则 | 示例 |
|------|------|
| 千分位逗号移除 | `1,234` → `1234` |
| 负百分比 | `-5.3%` → `负百分之五点三` |
| 百分数 | `12.5%` → `百分之十二点五` |
| 数字转中文 | `129.80` → `一百二十九点八零` |
| 年份逐字读 | `2025年` → `二零二五年` |
